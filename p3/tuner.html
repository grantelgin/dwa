In tune bookmarklet

Guitars can get out of tune easily and it's important to have a tuner near by. The In tune bookmarklet will be a Javascript based app the user can launch from their bookmark bar. The tuner will open in a small window over what ever web page the user is currently on and will not take them away from their current page. This is useful if you are trying to learn a song on youtube or reading tablature, you can stay on that page and tune your guitar at the same time.





Related examples

HTML 5 outliner is an example of a javascript bookmarklet that can load over any web page using position:fixed and z-index of 999.


Guituner.com uses jQuery to animate the icons and is an example of another web based guitar tuner.


Guitar tuner is a tutorial that outlines how to build a Javascript guitar tuner and gives examples of how to load the audio to play the correct notes.
